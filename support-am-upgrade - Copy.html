  1. Download the self-extracting file, AppManager8_2.exe.
  2. Copy the self-extracting file to the computer where you want to extract its contents.
  3. Run the AppManager8_2.exe program and extract the contents to a local directory. Do not extract the contents to a writable UNC path.

Notes:

  • AppManager 8.2 includes fixes and enhancements to all components of AppManager, including the repository, Management Server, Managed Client (or agent), Operator Console, and Web Console. The AppManager central components (repository, Management Server, and consoles) must all be upgraded to AppManager 8.2 at the same time. You can stage updates to your managed clients as best suits your schedule, because the AppManager 8.2 central components also support some previous versions of AppManager. For more information about upgrading, see the AppManager Upgrade and Migration Guide.
  • If you are having issues downloading the upgrade, please contact Tech Support to assist you.
